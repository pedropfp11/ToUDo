O processo de desenvolvimento do layout e da estrutura do site seguiu o seguinte fluxo de ferramentas:

*   **Prototipagem:** Utilizou-se o [Whimsical / Figma] como ferramenta para o desenvolvimento do layout de média fidelidade e mapeamento da experiência do usuário.
*   **Identidade Visual:** A definição da paleta de cores, tipografia e elementos gráficos foi estruturada através do [Google Stitch / Figma].
*   **Desenvolvimento:** A criação das páginas e funcionalidades do site foi realizada utilizando o [Google AI Studio / VS Code].

## Para Rodar Localmente:

*   Clone este repositório e acesse o diretório do projeto via terminal.

**Pré-requisitos:** Node.js instalado.

1. Instale as dependências do projeto:
   `npm install`
2. Inicie o servidor local para rodar o site:
   `npm run dev`
3. Abra o seu navegador de internet e insira a URL informada no terminal (ex: `http://localhost:5173`).
